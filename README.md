# 💬 Conversaciones en Inglés

Aplicación web para aprender inglés con pronunciación en español, diálogos interactivos y audio de texto a voz.

## 📱 Características

- 🔊 Audio Text-to-Speech en inglés americano
- 💬 Diálogos por escenarios (rutina, trabajo, viajes, médico, etc.)
- 🔤 Abecedario y números con pronunciación
- 🛡️ Protección anti-publicidad integrada
- 📱 Optimizado para móvil y WebView APK

---

## 🚀 Cómo usar en GitHub Pages

1. Sube este repositorio a GitHub
2. Ve a **Settings → Pages**
3. En *Source* selecciona **main** y carpeta **/ (root)**
4. Clic en **Save**
5. Tu app estará en: `https://TU-USUARIO.github.io/NOMBRE-REPO/`

---

## 📦 Cómo convertir en APK con AppGeyser

1. Ve a [appgeyser.com](https://appgeyser.com)
2. Selecciona **"Website"**
3. Pega la URL de tu GitHub Pages: `https://TU-USUARIO.github.io/NOMBRE-REPO/`
4. Ponle nombre, ícono y color a tu app
5. Clic en **Create** → descarga el APK
6. Instala en Android activando *"Orígenes desconocidos"*

## 🛡️ Protección anti-publicidad

La app incluye protección en 3 capas:
- **CSP** (Content Security Policy) — bloquea recursos externos no autorizados
- **JavaScript** — intercepta fetch/XHR/popups hacia dominios de ads
- **MutationObserver** — elimina elementos de publicidad del DOM en tiempo real
